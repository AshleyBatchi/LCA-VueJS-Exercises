## Project Overview

This project is a **Cooking Masterclass Catalogue** built using **Vue 3 + Vite**.

It is a single-page application that allows users to browse cooking courses, view chef details, see pricing, and save courses to a wishlist. It also includes filtering functionality to show only available courses.

This project is part of the Life Choices Academy frontend development training.

---

## Features

- Display cooking courses dynamically
- Show chef name, level, and price
- Sold-out status indicator
- Wishlist system (adds selected courses)
- Wishlist counter in header
- Filter available courses only (stretch goal)
- Responsive layout using Vue reactivity
